HTML Tags

Тег Предназначение тега
<html> Тег содержит внутри себя html страницу.
<!DOCTYPE> обьявляет тип документа, также может указывать на версиию языка разметки
<!--...--> ctrl ? Используеться для добовлений комментариев.
<head> Тег с основными свойствами html страницы.
<body> тег в котором находиться все отображаемые элементы на странице.
<title> Тег для определения заголовка.
<link> Одиночный тег для подключений внешних таблиц стилей.
<h1> - <h6> Заголовки шести уровней (по убывание размера)
<a> Он создает гипертекстовые ссылки
<img> Одиночный тег который вставляет изоброжение на стрницу с помощью атрибута src, значение котрого являеться адрес изоброженя
<color> Цвет текста
<main> используется для обозначения основного содержимого страницы.
<table> определяет HTML таблицу для представления табличных данных.
<tr> определяет строки в таблице. Ячейки в строке задаются тегами < th> (заголовочная ячейка) или < td>< /td>
<td> Тег < td> используется для создания стандартной ячейки HTML таблицы. Тег должен находиться внутри контейнера< tr>< /td>
<th> применяется в таблицах внутри тегов < tr> для создания заголовочной ячейки. Тескст введенный в заголовочную ячейку по умолчанию отображается жирным шрифтом и выравнивается по центру ячейки.
<colgroup> предназначен для задания ширины и стиля одной или нескольких колонок таблицы.
